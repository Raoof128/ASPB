# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-11-20

### Added
- Initial release of the Azure Service Principal Secret Expiry Bot.
- Feature: Authenticate via `DefaultAzureCredential`.
- Feature: Enumerate all Service Principals via Microsoft Graph API.
- Feature: Analyze Client Secrets and Certificates for expiry.
- Feature: CLI with support for Table, CSV, JSON, and Markdown outputs.
- Feature: "Critical Only" mode for filtering.
- Comprehensive unit tests.
- Production-ready documentation and repository structure.
